# 🩺 Smart Health Care Chatbot

A desktop-based **Smart Health Care Chatbot** built using **Python**, **Tkinter**, and **spaCy**. The chatbot uses Natural Language Processing (NLP) to analyze user symptoms, predict possible diseases, and provide basic remedies and precautions through an interactive graphical interface.

> **Disclaimer:** This project is for educational purposes only and should not be used as a substitute for professional medical advice.

---

# ✨ Features

- 🧠 Symptom analysis using Natural Language Processing (NLP)
- 💬 Interactive chatbot interface
- 👤 Personalized user greeting
- 🩺 Predicts possible diseases based on symptoms
- 💊 Suggests treatments
- ⚠️ Displays precautions
- 🎨 Clean and user-friendly GUI
- ⚡ Fast local execution

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Smart-Health-Care-Chatbot.git

cd Smart-Health-Care-Chatbot
```

## 2. Install Dependencies

```bash
pip install spacy
```

## 3. Download the English NLP Model

```bash
python -m spacy download en_core_web_sm
```

---

# ▶️ Running the Application

```bash
python app.py
```

---

# 💬 Usage

1. Launch the application.
2. Enter your name.
3. Click **Start Chat**.
4. Describe your symptoms naturally.
5. Receive:
   - Possible Disease
   - Suggested Treatment
   - Recommended Precautions

---

# 🩺 Supported Diseases

Currently, the chatbot can recognize symptoms related to:

- Fever
- Common Cold
- Migraine
- Food Poisoning
- Body Pain
- Weakness

Additional diseases can be added by extending the disease database in **bot.py**.

---

# 💡 Example Conversation

**User**

```text
I have fever and headache.
```

**Bot**

```text
🩺 Possible Condition: Fever

💊 Solution:
Take paracetamol, drink plenty of fluids and rest.

⚠️ Precautions:
Avoid cold drinks and monitor your temperature.
```
